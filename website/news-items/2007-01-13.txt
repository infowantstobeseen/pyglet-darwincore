date=2007-01-13

We've made some good progress in implementing the event framework for
scene2d. Map cells highlight when hovered, sprites are clickable, etc.
See the RECT_FLAT_MOUSE test for a demo. More info on <a
href="http://www.mechanicalcat.net/richard/log/Python/Another_pyglet_update">my
blog</a>. The highlighting uses a new
API (very pre-alpha) for adding effects to drawable things.
